# Seattle Airbnb Market Analysis Dashboard | Tableau

## Overview

This project analyzes Seattle Airbnb market data using Tableau. The dashboard provides insights into pricing trends, revenue growth, bedroom distribution, and geographic performance across zip codes.

## Tools Used

- Tableau
- Microsoft Excel
- Data Visualization
- Dashboard Design
